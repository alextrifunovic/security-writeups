# Resources

Curated list of useful resources I've found along the way.

## Learning Platforms

- [TryHackMe](https://tryhackme.com) — beginner-friendly, guided rooms
- [HackTheBox](https://www.hackthebox.com) — harder, more realistic machines
- [PortSwigger Web Security Academy](https://portswigger.net/web-security) — best free resource for web app security
- [PicoCTF](https://picoctf.org) — CTF challenges for all levels

## Reference Material

- [OWASP Top 10](https://owasp.org/www-project-top-ten/) — must-know web vulnerabilities
- [OWASP Cheat Sheet Series](https://cheatsheetseries.owasp.org/) — concise mitigation guides
- [HackTricks](https://book.hacktricks.xyz/) — encyclopedia of pentesting techniques
- [PayloadsAllTheThings](https://github.com/swisskyrepo/PayloadsAllTheThings) — payload collection

## Tools

- `nmap` — network scanning
- `Burp Suite` — web app testing proxy
- `Wireshark` — packet analysis
- `gobuster` / `ffuf` — directory and file fuzzing
- `Hydra` — password brute forcing
- `Metasploit` — exploitation framework
